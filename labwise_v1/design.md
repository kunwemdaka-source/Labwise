# Labwise Website Design Style Guide

## Design Philosophy

**Color Theme**: Purple-based color palette inspired by the Labwise brochure, creating a professional healthcare aesthetic that conveys trust, innovation, and compassion.

**Typography**: Modern, clean sans-serif fonts for optimal readability in healthcare contexts, with strategic use of bold weights for emphasis.

**Visual Language**: Professional medical facility aesthetic with warm, welcoming touches that emphasize patient-centered care and modern healthcare excellence.

## Color Palette

- **Primary Purple**: #6B46C1 (Deep purple for headers and key elements)
- **Secondary Purple**: #A78BFA (Lighter purple for accents and highlights)
- **Background**: #FAFAFA (Clean white-gray for content areas)
- **Text Dark**: #1F2937 (Dark gray for primary text)
- **Text Light**: #6B7280 (Medium gray for secondary text)
- **White**: #FFFFFF (Pure white for cards and containers)

## Typography

- **Display Font**: Inter (Bold weights for headings)
- **Body Font**: Inter (Regular and medium weights for content)
- **Accent Font**: Inter (Light weights for subtle elements)

## Visual Effects

- **Hero Background**: Subtle gradient overlay with purple tones
- **Cards**: Clean white backgrounds with subtle shadows
- **Buttons**: Purple gradient backgrounds with hover effects
- **Images**: Professional medical imagery with consistent filtering
- **Animations**: Smooth transitions and subtle hover effects

## Layout Principles

- **Grid System**: 12-column responsive grid
- **Spacing**: Consistent 8px base unit system
- **Containers**: Max-width containers with centered alignment
- **Sections**: Clear visual hierarchy with proper content separation
- **Mobile-First**: Responsive design starting from mobile breakpoints
